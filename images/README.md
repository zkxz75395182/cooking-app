├── avatars/             # 用戶頭像
│   ├── user1.jpg        # 用戶1頭像

│   └── user3.jpg        # 用戶3頭像
├── posts/               # 社區分享圖片

│   ├── fish2.jpg        # 魚類分享圖2

    ├── community.png    # 社區圖標
    ├── community-active.png

## 圖片規格

### 菜譜圖片
- 尺寸：400x300px
- 格式：JPG
- 質量：80%

### 用戶頭像
- 尺寸：120x120px
- 格式：JPG
- 圓形裁剪

### 社區分享圖片
- 尺寸：300x300px
- 格式：JPG
- 正方形裁剪

### 導航欄圖標
- 尺寸：48x48px
- 格式：PNG
- 透明背景

## 注意事項

1. 所有圖片應為原創或獲得授權
2. 圖片大小應控制在合理範圍內
3. 建議使用CDN加速圖片加載
4. 支持圖片懶加載優化性能
